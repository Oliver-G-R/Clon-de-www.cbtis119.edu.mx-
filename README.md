# Clone de http://www.cbtis119.edu.mx/

## Finalidad del proyecto
Este proyecto se hizo con la finalidad de practicar el diseño responsivo, como también practicar  
la metodología **BEM** y el preprocesador **SASS**.  

## Estructura del proyecto  
Cada parte de este proyecto esta estructurado por sus partes que lo conforman.  

1. Heder
2. Nav
3. Slider
4. Header
5. Sections
6. Main-content
7. Sidebar
8. Footer
9. Main (Este archivo es el que ejecuta todos los estilos sass)

## Fuentes externas utilizadas 
[Normalize](https://necolas.github.io/normalize.css/) Se uso para formatear los estilos por defecto del navegador
